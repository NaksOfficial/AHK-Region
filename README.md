# Биндер для статей УК, КоАП и УПК

**Версия 2.0**

**Разработчик: Виктор Смирнов**

**Последнее обновление: 02.04.2026 г.**

---

## Описание

Этот скрипт на AutoHotkey v2.0 предназначен для быстрого поиска и добавления статей трёх кодексов:

- **Уголовный кодекс (УК)** — с расчётом срока лишения свободы (максимальный срок по статье)

- **Кодекс об административных правонарушениях (КоАП)** — с расчётом срока ареста (в сутках)

- **Уголовно-процессуальный кодекс (УПК)** — для справки, без расчёта сроков

Скрипт позволяет:

- Искать статьи по номеру или названию

- Добавлять статьи в сохранённый список

- Автоматически рассчитывать суммарный срок (для УК и КоАП отдельно)

- Сохранять статьи в избранное

- Настраивать текст жетона

- Отображать оверлей с картой (F9)

- **Автоматически обновляться через GitHub**

---

## Горячие клавиши

| Клавиша | Действие |

|---------|----------|

| **F3** | Показать/скрыть все окна (поиск, сохранённые, звёздочки) |

| **F8** | Открыть окно настроек (описание и текст жетона) |

| **Alt+1** | Отправить текст жетона в чат |

| **Alt+X** | Отправить текущее время в чат |

| **F9** | Показать/скрыть оверлей с картой (map.png) |

---

## ️ Интерфейс

### Окно 1 (слева) — поиск статей

- Выбор кодекса (УК / КоАП / УПК)

- Поле поиска (фильтрация статей по номеру или названию)

- Список статей текущего кодекса

### Окно 2 (центр) — сохранённые статьи

- Список добавленных статей с полным описанием

- **Очистить список** — удаляет все статьи

- **Копировать** — копирует весь список в буфер обмена

### Окно 3 (справа) — вменяемые статьи

- Список добавленных статей с указанием кодекса

- Суммарный срок (для УК — в годах, для КоАП — в сутках, если есть оба — приоритет у УК)

---

## ⚙️ Настройки (F8)

- **Вкладка «Описание»** — информация о горячих клавишах и работе со статьями

- **Вкладка «Настройки»** — поле для ввода текста жетона (будет отправляться по Alt+1 без префикса /do, если он не начинается с /todo или /me)

## 🔄 Автоматическое обновление

Программа поддерживает автоматическое обновление через GitHub. Все обновления являются **обязательными** — при выходе новой версии обновление происходит автоматически.

### Как работает обновление:

1. При запуске `AHK.exe` проверяет файл `version.txt` на GitHub

2. Если версия на GitHub отличается от текущей, программа показывает сообщение и запускает `Updater.exe`

3. `Updater.exe` скачивает новую версию `AHK.exe` и заменяет старый файл

4. После успешного обновления программа закрывается — пользователь запускает её заново самостоятельно

| Файл | Назначение |

|------|------------|

| `AHK.exe` | Основной скрипт |

## Требования

- **AutoHotkey v2.0** (скрипт несовместим с v1.1)

- Операционная система Windows

- Изображение `map.png` в папке со скриптом (для работы F9)

- **Интернет-соединение** (для проверки обновлений)

---

## Как использовать

1. Запустите скрипт (двойным кликом по `.exe` файлу)

2. Нажмите **F3**, чтобы показать окна (если они не появились автоматически)

3. Выберите кодекс в выпадающем списке

4. Введите поисковый запрос (номер статьи или название)

5. Дважды кликните по нужной статье, чтобы добавить её в сохранённый список

6. В окне справа появится список добавленных статей и суммарный срок

7. Настройте текст жетона через **F8** и отправляйте его в чат по **Alt+1**

---

## ⚠️ Важные замечания

- **Суммарный срок** считается по максимальному сроку наказания, указанному в статье

- Если в списке есть статьи УК и КоАП, суммарный срок показывается только по УК (в годах)

- Статьи УПК не влияют на суммарный срок

- Для корректной работы оверлея (F9) файл `map.png` должен лежать в той же папке, что и скрипт

- При выходе новой версии обновление происходит автоматически — отменить его нельзя
